# Ambient Player

노마드코더 개발자 모임 2주차   

## Requirements

백색소음 또는 음악을 Background에서 재생해주는 사이트   
- 테마 : 소리 (<audio> 태그 필수)
- 대상 : 개발자, 3D Modeler, Artist, 작가 등 컴퓨터를 이용해 작업하는 사람들

## Reference

- https://rainymood.com/
- https://asoftmurmur.com/

## Concept

디즈니 캐릭터 이미지 클릭 시 해당 ost 재생

## 기능

- 캐릭터 썸네일 이미지 클릭 시 해당 음악의 처음부터 재생
- 음악에 맞게 배경 이미지 변경
- 일시정지 버튼
- 볼륨 조절
